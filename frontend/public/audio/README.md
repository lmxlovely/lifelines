# 背景音乐设置

请将音乐文件放置在此目录，需要准备**两首**音乐：

## 📁 文件说明

### 1. `story-bgm.mp3` - 普通背景音乐
- **触发时机**：任何故事开始播放时
- **适用场景**：所有用户查看人生轨迹时的背景音乐
- **推荐风格**：轻柔、舒缓的钢琴曲或氛围音乐
- **音量**：30%（代码中设置）

### 2. `reunion-bgm.mp3` - 彩蛋专属音乐 💕
- **触发时机**：仅当输入"李彦"和"李梦祥"，故事播放到最后重逢时
- **适用场景**：彩蛋的浪漫重逢特效
- **推荐风格**：浪漫、感人的情歌或钢琴曲
- **音量**：50%（代码中设置）

## ⚠️ 注意事项

- 请确保使用**免版权**或**已授权**的音乐
- 文件格式需为 **MP3**
- 建议文件大小控制在 **5MB 以内**以确保加载速度
- 选择适合**循环播放**的曲目
